# TuAcopioOrganizado

TuAcopioOrganizado es una aplicación descentralizada hecha por Analogic LLC, en colaboración con estudiantes de la UDO y del Club de Robótica de la U.E. Colegio Eduardo Blanco, Novu Maker 3D academy, diseñada para ayudar a administrar los centros de acopio.



## Indice
- [Prerequisitos](#Prerequisitos)
- [Instalación](#Instalación)
  
## Prerequisitos
Para ejecutar esta aplicación, asegúrate de tener instalado lo siguiente:
- **[Git](https://git-scm.com/install/)**: Sistema de control de versiones.
- **[HTML5](https://developer.mozilla.org/es/docs/Web/HTML)**: Navegador web moderno compatible con HTML5. (ej: Chrome o Firefox)

## Instalación
1. Clonar el repositorio
      ```bash
   git clone https://github.com/JD277/TuAcopioOrganizado.git
   ```
2. Ejecutar el archivo index.html

## Uso
La aplicación TuAcopioOrganizado funciona directamente desde un navegador web. Tras ejecutar el archivo index.html se va a mostrar una ventana con la siguiente información: 
<img width="1279" height="698" alt="pagina-principal" src="https://github.com/user-attachments/assets/adb17300-f714-4cd9-9202-88a66229dc08" />

Este es el "Panel de Administración" de la aplicación. Desde aquí se puede gestionar todo el inventario de productos y donaciones. Para registar una nueva donación se debe oprimir el botón azul "Ir al Registro" en la parte superior derecha del Panel de Administración. 

<img width="1279" height="698" alt="pagina-principal-iralregistro" src="https://github.com/user-attachments/assets/63aff666-68d2-4fdb-aa45-f7d5514c4ab8" />

La ventana de "Registro" está separada en 3 partes: 
* Superior: Se disponen de dos botones, uno para la ventana de Solicitudes, y otro para regresar al Panel de Administración.
* Medio: En donde se localiza el botón "Nueva Donación" para añadir las donaciones que se registren.
* Inferior: Se pueden observar las últimas donaciones realizadas, así como información de la donación en particular.

<img width="1279" height="694" alt="pagina-registro1" src="https://github.com/user-attachments/assets/3143784b-9d7b-41f8-a1de-43838a2d40d3" />

Al dar click al botón "Nueva Donación" se va a desplegar una mini ventana de opciones nueva, tal como se muestra a continuación: 

<img width="515" height="625" alt="registrar-nueva-donación" src="https://github.com/user-attachments/assets/1cbb8aed-cca2-4fd9-b179-01d169749ebe" />

En esta ventana se peude elegir la **categoría** (desde el Panel de Administración se pueden añadir más categorías), la cantidad y unidad de la donación, añadir la persona o institución donante, añadir una imagen del producto (opcional) y notas adicionales (opcional). Por último, se puede cancelar el registro de la donación, o guardar la donación. Tras añadir la donación, aparecerá en la parte inferior en las últimas donaciones realizadas.
